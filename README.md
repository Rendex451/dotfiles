# Коллекция моих дотфайлов

## Компоненты

- common - конфиги общие для всех nodes (рабочих машин)
- node-X - конфиги для определённой рабочей машины
- server - сборка конфигов для установки на сервера

## Установка

```bash
# выполняем из папки dotfiles
stow -v -t ~ common
stow -v -t ~ node-<X>

# для удаления имеющихся ссылок
stow -D -v common
```
